# Text-2-Beluga — Text to Discord-style Video

Generate short Discord-style conversation videos from a simple text script. The project converts text lines into rendered message images, compiles them into an MP4 with timing, and mixes sound effects and background music.

This repository is intended as a small, local tool for making short chat videos — useful for demos, tutorials, or social media posts.

---

## Quick summary

- Input: a plain text script (see `assets/example/example_script.txt`).
- Output: `final_video.mp4` in the repository root (generated by `scripts/compile_images.py` + `scripts/sound_effects.py`).
- Intermediate frames: written to the `chat/` directory during generation.
- Notable implemented features: strikethrough, URL support, mentions, bold/italic, emoji rendering, per-message durations, `#!sound` effects, `BGM:` background music, a typing indicator, unknown-character fallback, and experimental animations (`fade-in`, `typewriter`).

## Prerequisites

- Python 3.9+ (3.10 recommended)
- FFmpeg available on PATH (for image->video compilation)
- Install Python requirements:

```bash
pip install -r requirements.txt
```

## Installation

1. Clone the repo and install deps:

```bash
git clone <repo-url>
cd Text-2-Beluga
pip install -r requirements.txt
```

2. Ensure `ffmpeg` is installed and on PATH.

3. Add/verify assets:
- Profile pictures: `assets/profile_pictures/perm/` and `assets/profile_pictures/characters.json`.
- Sound files (`.mp3`) in `assets/sounds/mp3/`.

---

## Script format (short)

- Comments: lines beginning with `#` are ignored.
- Join: `WELCOME Name$^<seconds>#!sound` (optional `#!sound`).
- Character declaration: `Name:` on its own line.
- Message line: `Message text$^<seconds>[#!sound][|animation]`.

Examples:

```
WELCOME Alex$^1.5#!join_sound
Alex:
Hey everyone!$^2
Sophie:
Welcome Alex!$^3#!cheer
```

Notes:
- Durations after `$^` are required for messages and control how long the frame lasts.
- Sound effects use `#!name` where `name.mp3` must exist under `assets/sounds/mp3/`.
- Background music can be added with `BGM: name` (or `MUSIC: name`).
- Animation tags may be set after metadata with `|fade-in` or `|typewriter` (experimental).

## Usage

1. Validate a script (optional):

```bash
python scripts/script_validator.py assets/example/example_script.txt
```

2. Generate video (interactive menu):

```bash
python scripts/main.py
```

3. Or run the helper (non-interactive) to regenerate the example:

```bash
python scripts/run_example.py
```

After generation the final file is `final_video.mp4` in the repo root.

---

## Implementation notes

- Images are rendered with PIL + Pilmoji for emoji support.
- Video assembly uses FFmpeg (concat) to build a temporary `output.mp4`, then MoviePy mixes audio tracks and writes `final_video.mp4`.
- Intermediate `output.mp4` is removed after audio mixing.
- Animations are currently implemented by duplicating frames with simple blends/masks (costly in disk size). Consider moving complex effects into FFmpeg filters for performance.

---

## Limitations & known issues

- Animation set is experimental (only `fade-in` and `typewriter` implemented).
- The tool assumes durations are present for every message; missing durations are flagged by the validator.
- Large scripts produce many PNG frames in `chat/` — consider cleaning that folder after generating a final video.
